# BG & Limits MVP

Локальное веб-приложение для первого этапа MVP: загрузка Excel-портфеля банковских гарантий, нормализация данных, расчёт банковских лимитов, базовый дашборд и экспорт отчётов.

## Что уже реализовано

- Загрузка Excel-файла с листами `БГ` и `Лимиты`.
- Нормализация русскоязычных названий колонок, сумм, дат, ставок и статусов.
- Автоматическое формирование справочников банков и поставщиков.
- Расчёт использованного, зарезервированного и свободного лимита банка.
- Флаги риска по лимитам: `LIMIT_OVERUSED`, `LOW_AVAILABLE_LIMIT`, `LIMIT_NOT_LOADED`.
- Проверки качества данных по реестру БГ.
- Streamlit-страницы: `Dashboard`, `Bank Limits`, `Guarantees`, `Suppliers`, `Export`.
- Встроенные демо-данные и кнопка скачивания шаблона Excel.
- Экспорт полного отчёта в Excel и отдельных таблиц в CSV.

## Как запустить

1. Установите Python 3.11+.
2. В папке проекта выполните:

```bash
python -m pip install -r bank_guarantee_tool/requirements.txt
```

3. Запустите приложение:

```bash
streamlit run bank_guarantee_tool/app.py
```

4. Откройте ссылку, которую покажет Streamlit, обычно это `http://localhost:8501`.
5. Можно сразу работать с демо-данными или загрузить свой Excel-файл.

## Формат Excel для первого этапа

Минимальный лист `БГ`:

| Номер БГ | Банк | Поставщик | ИНН поставщика | Юрлицо | Сумма БГ | Дата начала | Дата окончания | Ставка | Комиссия | Статус |
|---|---|---|---|---|---:|---|---|---:|---:|---|

Опциональный лист `Лимиты`:

| Банк | Юрлицо | Общий лимит | Резерв | Статус лимита |
|---|---|---:|---:|---|

Если лист `Лимиты` не загружен, приложение всё равно покажет реестр БГ и поставщиков, но лимиты будут отмечены флагом `LIMIT_NOT_LOADED`.
