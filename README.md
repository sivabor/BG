# BG & Limits MVP

Локальное веб-приложение для первого этапа MVP: загрузка Excel-портфеля банковских гарантий, нормализация данных, расчёт банковских лимитов, базовый дашборд и экспорт отчётов.

## Что уже реализовано

- Загрузка Excel-файла с листами `БГ` и `Лимиты`.
- Нормализация русскоязычных названий колонок, сумм, дат, ставок и статусов.
- Автоматическое формирование справочников банков и поставщиков.
- Расчёт использованного, зарезервированного и свободного лимита банка.
- Флаги риска по лимитам: `LIMIT_OVERUSED`, `LOW_AVAILABLE_LIMIT`, `LIMIT_NOT_LOADED`.
- Проверки качества данных по реестру БГ.
- Streamlit-страницы: `Dashboard`, `Bank Limits`, `Guarantees`, `Suppliers`, `Export`.
- Встроенные демо-данные и кнопка скачивания шаблона Excel.
- Экспорт полного отчёта в Excel и отдельных таблиц в CSV.



## Проверка через HTML без установки

Если нужно просто открыть и проверить концепцию в браузере без Python, Streamlit и установки библиотек, используйте HTML-файл:

```text
Проверить_BG_Limits_HTML.html
```

Самый простой способ на Windows:

```text
Открыть_HTML_проверку_Windows.bat
```

Что умеет HTML-проверка:

- открывается двойным кликом как обычная веб-страница;
- содержит демо-данные по банковским гарантиям и лимитам;
- считает общий, использованный и свободный лимит;
- показывает риск-флаги по лимитам;
- показывает реестр гарантий и ближайшие окончания;
- позволяет загрузить Excel `.xlsx`, вставить CSV или загрузить CSV для быстрой проверки своих данных;
- показывает индикатор загрузки и обработки файла;
- показывает, какие строки Excel обработаны, а какие требуют ручной проверки.

Ограничение: HTML-страница — это демонстрационный режим для проверки интерфейса и расчётов. Загрузка Excel в HTML использует браузерный Excel-парсер SheetJS из CDN, поэтому для первой загрузки Excel-режима может понадобиться интернет. Полный MVP с серверной обработкой Excel `.xlsx` запускается через `Запустить_BG_Limits_Windows.bat` или команду `streamlit run bank_guarantee_tool/app.py`.

## Самый простой запуск на Windows

Если вы не технический специалист, используйте файл:

```text
Запустить_BG_Limits_Windows.bat
```

Как пользоваться:

1. Скачайте/распакуйте проект в обычную папку Windows.
2. Дважды щёлкните по файлу `Запустить_BG_Limits_Windows.bat`.
3. При первом запуске файл сам подготовит локальную среду в папке `.windows_runtime`.
4. Дождитесь открытия браузера с адресом `http://localhost:8501`.
5. В приложении можно сразу включить демо-данные или загрузить свой Excel. При загрузке Excel в боковой панели появится индикатор обработки.
6. Если файл распознался неполностью, откройте раздел `Import Diagnostics`: там видны все листы, включая скрытые, строка найденных заголовков, обработанные строки и строки, требующие проверки.

Важно: это не устанавливает приложение в Windows. Все технические файлы лежат рядом с проектом в папке `.windows_runtime`; если нужно полностью удалить приложение, просто удалите папку проекта. Для первого запуска может понадобиться интернет, чтобы скачать Python и библиотеки, если Python уже не установлен на компьютере.

## Как запустить

1. Установите Python 3.11+.
2. В папке проекта выполните:

```bash
python -m pip install -r bank_guarantee_tool/requirements.txt
```

3. Запустите приложение:

```bash
streamlit run bank_guarantee_tool/app.py
```

4. Откройте ссылку, которую покажет Streamlit, обычно это `http://localhost:8501`.
5. Можно сразу работать с демо-данными или загрузить свой Excel-файл.

## Формат Excel для первого этапа

Минимальный лист `БГ`:

| Номер БГ | Банк | Поставщик | ИНН поставщика | Юрлицо | Сумма БГ | Дата начала | Дата окончания | Ставка | Комиссия | Статус |
|---|---|---|---|---|---:|---|---|---:|---:|---|

Опциональный лист `Лимиты`:

| Банк | Юрлицо | Общий лимит | Резерв | Статус лимита |
|---|---|---:|---:|---|

Если лист `Лимиты` не загружен, приложение всё равно покажет реестр БГ и поставщиков, но лимиты будут отмечены флагом `LIMIT_NOT_LOADED`.
